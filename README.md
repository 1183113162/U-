## 使用说明

$\bigstar$ 使用前请先自带selenium库和相应的chromedriver

### 获取sid

本人比较懒还请你们手动获取sid（以后可能会加上这个功能。。

* 在课程主页面按下F12
* ctrl+f 在下面搜索框中搜索sid
* 复制相应的数字即可

### 配置文件

开始会要你输入你的用户名，密码，sid来进行文件的配置之后是默认你输入的配置

如果你想改配置：

* 可以在config.ini里面直接改
* 在程序里按照提示输入相应指令（不建议
